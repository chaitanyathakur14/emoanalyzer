# Emotions Assessment Platform

## Overview
Emotions Assessment Platform is an AI-powered real-time emotion recognition and personality assessment system developed using Computer Vision, Deep Learning, and NLP techniques. The platform analyzes live camera feed data to detect facial emotions, evaluates behavioral patterns, and provides psychological feedback through interactive assessment modules and AI-driven interaction.

---

## Features

### Real-Time Emotion Detection
- Real-time facial emotion recognition using CNN and OpenCV
- Detects 7 emotional categories:
  - Happy
  - Sad
  - Angry
  - Fear
  - Surprise
  - Neutral
  - Disgust
- Live webcam feed processing with emotion prediction overlays

### Personality Assessment Module
- Interactive personality and psychological assessment system
- Generates feedback based on:
  - emotional responses
  - behavioral patterns
  - visual object interpretation
- Dynamic scoring and emotional analysis

### AI Chat Assistant
- Integrated AI chat feature for user interaction
- Uses manually curated Mumbai-specific datasets
- Supports localized conversational interaction
- Secure user data handling and storage

### Gamified Cognitive Assessment
- Interactive games designed to evaluate:
  - emotional resilience
  - decision-making behavior
  - cognitive response patterns
- Behavioral analytics through user interaction tracking

## System Workflow

1. Capture real-time webcam feed
2. Detect face using OpenCV
3. Process image frames using CNN model
4. Predict emotion category
5. Generate emotion analytics
6. Run personality assessment modules
7. Enable AI chat interaction
8. Store and analyze user behavioral data
